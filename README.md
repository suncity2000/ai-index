# AI Analysis Index

AI 모델들의 성능을 종합적으로 분석하고 비교하는 대시보드입니다.

## 📊 주요 기능

- **5가지 AI 카테고리 분석**
  - LLM (Large Language Models)
  - Text-to-Image
  - Text-to-Speech
  - Text-to-Video
  - Image-to-Video

- **LLM 다차원 평가**
  - 종합 순위
  - 코딩 성능
  - 수학 성능
  - 가성비
  - 속도

- **한국 AI 서비스 추적**
  - 한국 회사의 AI 모델 표시
  - 국내 서비스 개수 통계

- **사용자 경험**
  - 다크모드/라이트모드 지원
  - 반응형 디자인
  - 실시간 데이터 업데이트 (매일 오전 9시)

## 🛠 기술 스택

- **Frontend**: HTML, CSS, JavaScript
- **Styling**: Tailwind CSS
- **Data Source**: Artificial Analysis API
- **Automation**: GitHub Actions
- **Hosting**: GitHub Pages

## 📅 데이터 업데이트

- 매일 한국시간 오전 9시 자동 업데이트
- GitHub Actions를 통한 자동화
